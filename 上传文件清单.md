## 上传方式

### 方式 1：使用 SFTP（推荐）

使用 WinSCP 或 FileZilla：
1. 连接到服务器
2. 导航到 `/root/ST-server`
3. 拖拽文件上传（覆盖原文件）

### 方式 2：使用 SCP 命令

```bash
# 从本地上传到服务器
scp database.js root@119.8.118.149:/root/ST-server/
scp routes/auth.js root@119.8.118.149:/root/ST-server/routes/
scp server.js root@119.8.118.149:/root/ST-server/
scp scripts/fix-online-status.js root@119.8.118.149:/root/ST-server/scripts/
scp scripts/check-database.js root@119.8.118.149:/root/ST-server/scripts/
scp scripts/check-config.js root@119.8.118.149:/root/ST-server/scripts/
scp package.json root@119.8.118.149:/root/ST-server/
```

### 方式 3：使用 Git

如果使用 Git 版本控制：
```bash
# 在服务器上
cd /root/ST-server
git pull
```
