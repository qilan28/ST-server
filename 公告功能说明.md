# 📢 公告功能说明

## 功能概述

系统新增公告管理功能，支持超级管理员创建和管理公告，支持在登录页和用户面板独立显示不同的公告。

## 功能特性

### 1. 管理员功能
- ✅ 创建公告（支持登录页和用户面板两种类型）
- ✅ 编辑公告（标题、内容、启用状态）
- ✅ 删除公告
- ✅ 启用/禁用公告
- ✅ 查看所有公告列表

### 2. 公告类型
- **登录页公告** (`login`): 显示在登录/注册页面顶部
- **用户面板公告** (`dashboard`): 显示在用户控制台顶部

### 3. 公告显示
- 自动加载最新的活跃公告
- 精美的渐变背景设计
- 显示标题、内容和发布时间
- 移动端自适应

## 使用指南

### 管理员操作

#### 1. 创建公告
1. 登录管理员账号
2. 进入管理员面板
3. 在"📢 公告管理"区域点击"➕ 创建新公告"
4. 填写信息：
   - **类型**: 选择"登录页公告"或"用户面板公告"
   - **标题**: 公告标题（最多100字）
   - **内容**: 公告内容（最多500字）
   - **立即启用**: 勾选后公告立即生效
5. 点击"保存"

#### 2. 编辑公告
1. 在公告列表中点击"✏️"按钮
2. 修改公告信息
3. 点击"保存"

#### 3. 启用/禁用公告
- 点击公告列表中的"▶️"（启用）或"⏸️"（禁用）按钮
- 禁用的公告不会在前端显示

#### 4. 删除公告
- 点击公告列表中的"🗑️"按钮
- 确认后删除（不可恢复）

### 前端显示

#### 登录页公告
- 位置：登录/注册表单上方
- 显示：最新的一条活跃"登录页公告"
- 样式：紫色渐变背景卡片

#### 用户面板公告
- 位置：用户控制台顶部
- 显示：最新的一条活跃"用户面板公告"
- 样式：紫色渐变背景卡片（更大尺寸）

## API 接口

### 管理员接口（需要认证 + 管理员权限）

#### 获取所有公告
```
GET /api/admin/announcements
```

#### 创建公告
```
POST /api/admin/announcements
Body: {
  "type": "login" | "dashboard",
  "title": "公告标题",
  "content": "公告内容"
}
```

#### 更新公告
```
PUT /api/admin/announcements/:id
Body: {
  "title": "公告标题",
  "content": "公告内容",
  "isActive": true | false
}
```

#### 切换公告状态
```
PATCH /api/admin/announcements/:id/toggle
```

#### 删除公告
```
DELETE /api/admin/announcements/:id
```

### 公共接口（无需认证）

#### 获取指定类型的活跃公告
```
GET /api/announcements/:type
参数: type = "login" | "dashboard"
```

## 数据库结构

```sql
CREATE TABLE announcements (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    type TEXT NOT NULL,              -- 公告类型: login, dashboard
    title TEXT NOT NULL,              -- 公告标题
    content TEXT NOT NULL,            -- 公告内容
    is_active INTEGER DEFAULT 1,     -- 是否启用: 1-启用, 0-禁用
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    updated_at DATETIME DEFAULT CURRENT_TIMESTAMP
);
```

## 注意事项

1. ⚠️ 每种类型只显示最新的一条活跃公告
2. ⚠️ 公告内容支持换行，但不支持HTML
3. ⚠️ 删除公告操作不可撤销
4. ⚠️ 公告标题最多100字，内容最多500字
5. ✅ 公告在登录页和用户面板是完全独立的
6. ✅ 支持移动端自适应显示

## 常见问题

**Q: 为什么看不到公告？**
A: 检查：
- 公告是否已启用（is_active = 1）
- 公告类型是否正确（login/dashboard）
- 浏览器是否缓存（试试强制刷新）

**Q: 可以创建多条公告吗？**
A: 可以创建多条，但前端只显示最新的一条活跃公告。

**Q: 公告支持富文本吗？**
A: 目前只支持纯文本，不支持HTML或Markdown。
