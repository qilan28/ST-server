# 📢 公告功能说明

## 功能概述

系统新增公告管理功能，支持超级管理员创建和管理公告，支持在登录页和用户面板独立显示不同的公告。

## 功能特性

### 1. 管理员功能
- ✅ 创建公告（支持登录页和用户面板两种类型）
- ✅ 编辑公告（标题、内容、启用状态）
- ✅ 删除公告
- ✅ 启用/禁用公告
- ✅ 查看所有公告列表

### 2. 公告类型
- **登录页公告** (`login`): 显示在登录/注册页面顶部
- **用户面板公告** (`dashboard`): 显示在用户控制台顶部

### 3. 公告显示
- **轮播展示**：多个公告自动轮播显示（每5秒切换）
- **手动切换**：左右箭头按钮手动切换公告
- **指示器**：圆点指示器显示当前公告位置，可点击快速跳转
- **精美设计**：渐变背景、圆滑动画
- 显示标题、内容和发布时间
- 移动端自适应

## 使用指南

### 管理员操作

#### 1. 创建公告
1. 登录管理员账号
2. 进入管理员面板
3. 在"📢 公告管理"区域点击"➕ 创建新公告"
4. 填写信息：
   - **类型**: 选择"登录页公告"或"用户面板公告"
   - **标题**: 公告标题（最多100字）
   - **内容**: 公告内容（最多500字）
   - **立即启用**: 勾选后公告立即生效
5. 点击"保存"

#### 2. 编辑公告
1. 在公告列表中点击"✏️"按钮
2. 修改公告信息
3. 点击"保存"

#### 3. 启用/禁用公告
- 点击公告列表中的"▶️"（启用）或"⏸️"（禁用）按钮
- 禁用的公告不会在前端显示

#### 4. 删除公告
- 点击公告列表中的"🗑️"按钮
- 确认后删除（不可恢复）

### 前端显示

#### 登录页公告
- **位置**：登录/注册表单上方
- **显示**：所有活跃的"登录页公告"（轮播显示）
- **样式**：紫色渐变背景卡片
- **交互**：
  - 自动轮播（5秒/次）
  - 点击左右箭头手动切换
  - 点击指示器圆点跳转到指定公告
  - 手动切换后重新启动自动轮播

#### 用户面板公告
- **位置**：用户控制台顶部
- **显示**：所有活跃的"用户面板公告"（轮播显示）
- **样式**：紫色渐变背景卡片（更大尺寸）
- **交互**：
  - 自动轮播（5秒/次）
  - 点击左右箭头手动切换
  - 点击指示器圆点跳转到指定公告
  - 手动切换后重新启动自动轮播

## API 接口

### 管理员接口（需要认证 + 管理员权限）

#### 获取所有公告
```
GET /api/admin/announcements
```

#### 创建公告
```
POST /api/admin/announcements
Body: {
  "type": "login" | "dashboard",
  "title": "公告标题",
  "content": "公告内容"
}
```

#### 更新公告
```
PUT /api/admin/announcements/:id
Body: {
  "title": "公告标题",
  "content": "公告内容",
  "isActive": true | false
}
```

#### 切换公告状态
```
PATCH /api/admin/announcements/:id/toggle
```

#### 删除公告
```
DELETE /api/admin/announcements/:id
```

### 公共接口（无需认证）

#### 获取指定类型的活跃公告
```
GET /api/announcements/:type
参数: type = "login" | "dashboard"
```

## 数据库结构

```sql
CREATE TABLE announcements (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    type TEXT NOT NULL,              -- 公告类型: login, dashboard
    title TEXT NOT NULL,              -- 公告标题
    content TEXT NOT NULL,            -- 公告内容
    is_active INTEGER DEFAULT 1,     -- 是否启用: 1-启用, 0-禁用
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP,
    updated_at DATETIME DEFAULT CURRENT_TIMESTAMP
);
```

## 常见问题

**Q: 为什么看不到公告？**
A: 检查：
- 公告是否已启用（is_active = 1）
- 公告类型是否正确（login/dashboard）
- 浏览器是否缓存（试试强制刷新）

**Q: 可以创建多条公告吗？**
A: 可以！创建多条同类型的公告，它们会以轮播形式全部显示。

**Q: 公告支持富文本吗？**
A: 目前只支持纯文本，不支持HTML或Markdown。但支持换行（`\n`会自动换行）。

**Q: 如何调整轮播速度？**
A: 默认每5秒切换一次。要修改速度，编辑 `auth.js` 和 `dashboard.js` 中的 `setInterval` 时间参数（单位：毫秒）。

**Q: 轮播会影响性能吗？**
A: 不会。轮播只是简单的定时器切换，对性能影响极小。

**Q: 单个公告时会显示轮播控制吗？**
A: 不会。只有当同类型公告≥2条时，才显示箭头按钮和指示器。
