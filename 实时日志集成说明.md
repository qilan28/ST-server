# 🔥 备份实时日志功能 - 集成说明

## 📋 功能说明

使用 **Server-Sent Events (SSE)** 技术实现备份过程的实时日志推送，让用户可以看到：
- 🗜️ 压缩进度
- 📥 克隆仓库状态
- 💾 Git LFS 配置
- 📋 文件复制进度
- 💬 Git 提交状态
- 🚀 推送进度
- ✅ 完成/失败状态

### 前端集成

#### 方式 1: 使用 EventSource API

```javascript
function startBackupWithLogs() {
    // 创建 SSE 连接
    const eventSource = new EventSource('/api/backup/backup', {
        withCredentials: true  // 发送认证 Cookie
    });
    
    // 监听消息
    eventSource.onmessage = (event) => {
        const data = JSON.parse(event.data);
        
        // 显示日志
        console.log(data.message);
        addLogToUI(data.message, data.type);
        
        // 检查完成状态
        if (data.type === 'done') {
            console.log('备份成功！', data.result);
            eventSource.close();
        } else if (data.type === 'error') {
            console.error('备份失败！', data.error);
            eventSource.close();
        }
    };
    
    // 错误处理
    eventSource.onerror = (error) => {
        console.error('连接错误:', error);
        eventSource.close();
    };
}
```

#### 方式 2: 使用 fetch API + ReadableStream

```javascript
async function startBackupWithLogs() {
    const response = await fetch('/api/backup/backup', {
        method: 'POST',
        credentials: 'include'
    });
    
    const reader = response.body.getReader();
    const decoder = new TextDecoder();
    
    while (true) {
        const { done, value } = await reader.read();
        if (done) break;
        
        const chunk = decoder.decode(value);
        const lines = chunk.split('\n');
        
        for (const line of lines) {
            if (line.startsWith('data: ')) {
                const data = JSON.parse(line.substring(6));
                console.log(data.message);
                addLogToUI(data.message, data.type);
            }
        }
    }
}
```

## ⚠️ 注意事项

### 1. 认证
SSE 请求需要携带认证信息：
```javascript
new EventSource('/api/backup/backup', {
    withCredentials: true  // 发送 Cookie
});
```

### 2. 错误处理
必须处理连接错误：
```javascript
eventSource.onerror = (error) => {
    console.error('连接错误:', error);
    eventSource.close();
    // 恢复按钮状态
};
```

### 3. 资源清理
备份完成后关闭连接：
```javascript
if (data.type === 'done' || data.type === 'error') {
    eventSource.close();
}
```

### 4. 超时设置
对于长时间备份，需要调整服务器超时：
```javascript
// 在 server.js 中
server.timeout = 600000;  // 10分钟
```
